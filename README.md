# JWST SolarSystem Pipelines


All the pipelines used for Solar System extended sources data. For Jupiter, its size, brightness and rotation speed make it a really challenging target. That is why the pipelines have to be modified in order to retrieve the best data quality. All these pipelines work for 1 dither data. The mosaicking is done after the pipeline process with the Level3 cubes.

The integrated desaturation pipeline has been updated (03/03/2026).

This was a collaboration between the JWST Solar System teams from Observatoire de Paris-Meudon (Pablo Rodriguez Ovalle and Thierry Fouchet) and Leicester University (Jake Harkett and Leigh Fletcher). For more information please go to Jake Harkett's Github:  https://github.com/JHarkett/MIRI-code
